## Import

```js
import { ChaseSpinner } from "@hrc/spinner";
```

## Props

**It receives the same [props of `<SpinnerBase />`](../spinner-base#props)**

## CSS Variables

**It uses the same [CSS Variables of `<SpinnerBase />`](../spinner-base#css-variables)**

## CSS Classes

**`.chase-spinner`**

The base styles for `<ChaseSpinner />` component.

## Default values

### Speed

#### Normal

**Value:** `2.5s`

#### Slow

#### Fast
