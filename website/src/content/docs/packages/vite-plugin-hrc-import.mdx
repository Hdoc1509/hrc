Automatically use path import for components of `<hrc />` packages to reduce bundle size.

:::note
If your project doesn't support [`Vite`](https://vitejs.dev/), you can use
[`@hrc/babel-config`](/hrc/packages/babel-config) instead.
:::

## Installation

<InstallCommand packages={[packageName]} isDev />

## Usage

In your `vite.config.js` file:

```js
import hrcImportPlugin from "vite-plugin-hrc-import";

export default {
  plugins: [hrcImportPlugin()],
};
```

## Supported packages

- [`@hrc/button`](/hrc/packages/button)
- [`@hrc/input`](/hrc/packages/input)
- [`@hrc/spinner`](/hrc/packages/spinner)
