Create Babel config for component packages of `<hrc />`

## Installation

<InstallCommand packages={["@hrc/babel-config", "babel-plugin-import"]} isDev />

## Usage

In your `.babelrc.js` file:

```js
import hrcConfig from "@hrc/babel-config";

// single package
const plugins = [
  ...otherPlugins,
  ...hrcConfig("input")
];

// multiple packages
const plugins = [
  ...otherPlugins,
  ...hrcConfig(["button", "input"])
];

// all supported packages
const plugins = [
  ...otherPlugins,
  ...hrcConfig("all")
];

module.exports = { plugins };
```

## Supported packages

- [`@hrc/button`](../button)
- [`@hrc/input`](../input)
- [`@hrc/spinner`](../spinner)
