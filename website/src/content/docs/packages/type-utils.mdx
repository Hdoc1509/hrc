## Installation

<InstallCommand packages={[packageName]} isDev />

## Usage

```ts
import type { Simplify } from "@hrc/type-utils";

type MyType = {
  a: number;
  b: string;
};

type AnotherType = {
  c: boolean;
  d: number;
};

type FlattenedType = Simplify<MyType & AnotherType>;
```

## Utilities

- `Simplify` - Use to flatten the type output to improve type hints shown in
  editors

## Common Props

- `Color` - Color variants
- `Size` - Size variants
