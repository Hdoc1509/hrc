Type utilities use by the following packages of `<hrc />`:

- [`@hrc/button`](/packages/button)
- [`@hrc/input`](/packages/input)

## Installation

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install @hrc/type-utils
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @hrc/type-utils
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @hrc/type-utils
    ```
  </TabItem>
</Tabs>

## Usage

```ts
import type { Simplify } from "@hrc/type-utils";

type MyType = {
  a: number;
  b: string;
};

type AnotherType = {
  c: boolean;
  d: number;
};

type FlattenedType = Simplify<MyType & AnotherType>;
```

## Utilities

- `Simplify` - Use to flatten the type output to improve type hints shown in
  editors

## Common Props

- `Color` - Color variants
- `Size` - Size variants
